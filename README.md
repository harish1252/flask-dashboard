# Expense Tracker Web Application

A full-stack Expense Tracker web application built using Flask, SQLAlchemy, and Flask-Login.

This project allows users to register, login, and manage their personal expenses securely.

---

## Features

-  User Registration & Login (Authentication)
-  Password Hashing (Secure Storage)
-  Add Expenses
-  Edit Expenses
-  Delete Expenses
-  Dashboard with Total Expense Calculation
-  Category-wise Expense Chart
-  User-specific data (Each user sees only their expenses)
-  Logout functionality

---

## Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite
- HTML
- CSS
